# ==================

# Functionalities

## unregistred user

- _read articles_
- _search articles by title_
- _search articles by category_
- _register_

## registred user

- _comment articles_
- _like/unlike/dislike comments_
- _edit profile_
- _view profile details_
- _delete comments_
- _login_
- _logout_

## admin

- _create articles_
- _upload images_
- _edit articles_
- _delete user_
- _login as admin_

## articles

- _top five by views_
- _latest articles_
- _most liked comments_
